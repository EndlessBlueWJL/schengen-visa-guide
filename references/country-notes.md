# 申根签申请国对比

> 核心原则：行程单配合申请国。申请国天数写最长，出入境尽量走申请国。

## 三国推荐

| | 法国 | 意大利 | 西班牙 |
|---|---|---|---|
| **签证中心** | TLScontact | VFS Global | BLS |
| **领区限制** | 不分领区，全国任意城市可递 | 分领区 | 分领区 |
| **出签风格** | 偏紧，给的天数基本=行程单天数 | 通过率不错，对学生友好 | 有时会比行程多给几天 |
| **适合人群** | 不在北上广的、想就近递签的 | 行程主要在意国的 | 想多蹭几天的 |

### 直链

**法国**（双系统）
1. 先在 France-Visas 填表获取 FV 编号：https://france-visas.gouv.fr/zh
2. 再到 TLScontact 预约递签：https://visas-fr.tlscontact.cn

**意大利**
- 预约网站：https://visa.vfsglobal.com/chn/zh/ita

**上海领区流程：**
1. 填完申请表 → 获取申请码，**截图保存**
2. 系统会发一封邮件到邮箱，内含**系统派发的账号和密码**
3. 等待系统审核通过
4. 审核通过后会再发一封邮件通知缴费
5. 用系统派发的账号密码登录缴费

**西班牙**
- 填表 + 预约：https://web.blscn.cn

## 其他国家

用户选了上述国家之外的申根国时，按以下流程：

1. 对照上表查该国对应的签证中心（VFS / TLS / BLS / 独立）
2. 确认是否有领区限制
3. 提醒：行程单里该国天数写最长

## 通用提醒

- 申根签是短期旅游签（C 签），最多停留 90 天
- 有申根签后可在申根区 29 国自由通行
- 第一次申请建议走法/意/西，通过率最高
- 不管真实计划怎么玩，材料上行程单、酒店、机票三者必须自洽
- 认准官方域名 `vfsglobal.com`、`tlscontact.cn`、`blscn.cn`，防假冒网站

## 预约没号怎么办

暑期（5-8 月）热门国家号源紧张，遇到没号：

1. **多刷** — 不是一次性放完，分批放的。每天 8:00、12:00、20:00 前后多刷新
2. **换邮箱重新注册** — 换个新邮箱再约一次，有时不同邮箱看到的可用时段不同
3. **看领区其他城市** — 同一领区内的城市可以跨着递（如上海领区下杭州没号试试上海、南京）
4. **VIP/优选时段** — VFS/TLS 有付费优选时段，号比普通的多
5. **别找黄牛** — 预约免费，不存在官方「代抢」。「加急预约」「100% 出签」都是骗钱的，材料造假后果是永久不良记录
